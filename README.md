# PngForge

> Batch image converter — convert PNG, JPG, WebP, and AVIF with drag-and-drop simplicity.

## Features

- Convert between PNG, JPG, WebP, and AVIF formats
- Batch process hundreds of images at once
- Drag-and-drop interface for quick file loading
- Adjust output quality and compression level per format
- Preserve EXIF metadata or strip it for privacy
- Resize and crop images during conversion
- Preview images before and after conversion
- Download individual files or a single ZIP archive
- Full offline support — no server, no uploads

## Why PngForge?

Your images never leave your device. PngForge runs entirely in the browser using WebAssembly-powered encoders, so every conversion happens locally. There are no file size limits, no upload queues, and no privacy concerns. It's free, fast, and works without an internet connection.
